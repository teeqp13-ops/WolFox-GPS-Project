# WolFox GPS KSA 🦊🇸🇦

أداة متقدمة لنظام iOS للتحكم في الموقع الجغرافي وبيانات الجهاز، مصممة باحترافية لتوفير أقصى درجات التحكم والأمان.

---

## ✨ المميزات (Features)

- 📍 **تغيير الموقع (GPS Spoofing)**: تغيير موقعك الجغرافي إلى أي مكان في العالم بدقة عالية.
- 🛡️ **تخطي الحماية (Jailbreak Bypass)**: حماية متقدمة لتخطي اكتشاف الجيلبريك في التطبيقات.
- 🆔 **تغيير بيانات الجهاز (Device ID Spoofing)**: إمكانية تغيير UDID و Device ID الخاص بجهازك.
- 📸 **تغيير بيانات الكاميرا**: دعم تغيير مدخلات الكاميرا والموقع في الصور.
- 📡 **محاكاة البلوتوث (BLE Spoofing)**: التحكم في بيانات الـ Beacon والبلوتوث المحيطة.

---

## 🛠️ كيفية التثبيت (Installation)

1. قم بتحميل ملف الـ `.deb` من قسم [Actions](../../actions) في هذا المستودع.
2. انقل الملف إلى جهاز الآيفون الخاص بك.
3. استخدم **Filza File Manager** لتثبيت الحزمة.
4. قم بعمل **Respring** للجهاز لتفعيل الأداة.

---

## ⚙️ الإعدادات (Settings)

يمكنك الوصول إلى لوحة التحكم الخاصة بالأداة من خلال:
`الإعدادات (Settings) -> WolFox GPS`
